How It Works

Your Content

When you're logged in and viewing content you created:

  • Green "You're the author" badge appears in the top-right
  • Light blue action area with 3 buttons: Read, Edit, Delete
  • Edit button opens the editor with your content pre-loaded
  • Delete button shows double confirmation before deleting

Other Users' Content

When viewing content created by others:

  • No author badge or special highlighting
  • Only "Read" button available
  • No edit or delete permissions

Admin Users (Future)

Site administrators could see Edit/Delete buttons on all content for moderation purposes.
