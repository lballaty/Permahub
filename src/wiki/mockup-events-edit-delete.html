How It Works

Your Events

When you're logged in and viewing events you organized:

  • Green "You're the organizer" badge appears next to the title
  • Light blue action area with Edit and Delete buttons
  • Edit button opens the event editor with pre-loaded data
  • Delete button shows confirmation before removing the event

Other Organizers' Events

When viewing events organized by others:

  • No organizer badge or special highlighting
  • Standard Register/Export buttons only
  • No edit or delete permissions

Event Management Features

Managing your events:

  • Edit - Update date, time, location, description, contact info
  • Delete - Remove canceled or past events
  • Status - Mark as draft, published, or archived in the editor
